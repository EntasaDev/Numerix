# Numerix (ENG)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
Numerix is a lightweight yet powerful application for exploring and applying numerical methods.
## Features

- Various numerical methods implementations
- Extensible plugin system
## Getting Started
### Prerequisites
- Python 3.8+
- pip
### Installation
```bash
git clone https://github.com/yourusername/numerix.git
cd numerix
pip install -r requirements.txt
python main.py
```
### Basic Usage

1. Enter your function using standard math notation
    
2. Set integration bounds
    
3. Select computation method
    
4. Specify desired precision
    
5. Click "Compute" or run the script
#### Supported Operators

| Operator | Description    |
| -------- | -------------- |
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |
| `^`      | Exponentiation |
#### Integration Variable

- All integrations are performed with respect to **x**
### Supported Functions

|Function|Description|
|---|---|
|`sin(x)`|Sine|
|`cos(x)`|Cosine|
|`tan(x)`|Tangent|
|`sinh(x)`|Hyperbolic sine|
|`cosh(x)`|Hyperbolic cosine|
|`tanh(x)`|Hyperbolic tangent|
|`asin(x)`|Arcsine|
|`acos(x)`|Arccosine|
|`atan(x)`|Arctangent|
|`asinh(x)`|Hyperbolic arcsine|
|`acosh(x)`|Hyperbolic arccosine|
|`atanh(x)`|Hyperbolic arctangent|
|`log(x)`|Natural logarithm|
|`exp(x)`|Exponential|
|`log10(x)`|Base-10 logarithm|
|`log2(x)`|Base-2 logarithm|
|`sqrt(x)`|Square root|
|`abs(x)`|Absolute value|

### Example Inputs

- `sin(x)*cos(x)`
    
- `(x^2 + 3*x - 5)/(x + 1)`
    
- `exp(-x^2)` (Gaussian function)
    
- `abs(sin(x)) + sqrt(cos(x^2))`

# Numerix (RUS)
Numerix - это легкое, но мощное приложение для исследования и применения численных методов.
## Возможности

- Реализации различных численных методов
- Расширяемая система плагинов
## Начало работы
### Требования

- Python 3.8+
- pip
### Установка
```bash
git clone https://github.com/yourusername/numerix.git
cd numerix
pip install -r requirements.txt
python main.py
```
### Базовое использование

1. Введите функцию в стандартной математической нотации
    
2. Установите границы интегрирования
    
3. Выберите метод вычисления
    
4. Укажите желаемую точность
    
5. Нажмите "Вычислить" или запустите скрипт
#### Поддерживаемые операторы

| Оператор | Описание             |
| -------- | -------------------- |
| `+`      | Сложение             |
| `-`      | Вычитание            |
| `*`      | Умножение            |
| `/`      | Деление              |
| `^`      | Возведение в степень |
|          |                      |
#### Переменная интегрирования
- Все вычисления выполняются по переменной **x**
